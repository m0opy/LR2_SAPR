# APB Slave — Testbench and Coverage

Проект реализует простое устройство **APB-slave** и тестбенч `TB`, моделирующий работу устройства с проверкой переходов состояния ("светофор") и покрытием функционала.

## Состав
- **apb_slave.sv** -- RTL-модуль APB-slave с внутренней FSM ("traffic light").
- **tb.sv** -- тестбенч для верификации работы DUT, включая позитивные и негативные сценарии.
- **run.do** -- скрипт запуска.
